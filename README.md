# 📰 News Headline Fetcher

A Python command-line app that fetches live top headlines 
by category using the NewsAPI.

## Features
- Choose from 7 news categories
- Displays top 10 headlines per category
- View full article description and link
- Live data pulled fresh every time

## How to Run
1. Get a free API key at newsapi.org
2. Paste your key into the API_KEY variable
3. Run: `python news.py`

## Built With
- Python 3
- NewsAPI (free tier)
